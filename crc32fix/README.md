# crc32fix

练手作 & 第一作

作用: 以爆破宽/高的方式修复 png 的 crc32 校验错误

```text
crc32fix 0.1
Aloxaf <aloxafx@gmail.com>
根据 crc32 校验值修复受损图片的宽高

USAGE:
    crc32fix [OPTIONS] <inputfile>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -o <outputfile>        输出文件

ARGS:
    <inputfile>    受损文件路径
```
