img2ascii
--

图片转字符画

```text
img2ascii 0.1
Aloxaf <aloxafx@gmail.com>
图片转字符画

USAGE:
    img2ascii [FLAGS] [OPTIONS] <filename>

FLAGS:
    -f, --force      当输出文件存在时不询问直接覆盖
        --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -h, --height <height>        字符画的高 (默认更随宽变化)
    -o, --output <outputfile>    输出文件
    -w, --width <width>          字符画的宽 (默认80)

ARGS:
    <filename>    图片完整路径

```
